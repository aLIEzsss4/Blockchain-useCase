# blockchain-usecase

To install dependencies:

```bash
bun install
```

To run:

```bash
bun start
```

To run tests:

```bash
bun test
```



## Task

Requirements: Use bun + viem to complete interactions with Uniswap or PancakeSwap project

```
1. ETH swap to USDT
2. USDT swap to ETH
3. Add ETH/USDT LP
4. Remove ETH/USDT LP
```
